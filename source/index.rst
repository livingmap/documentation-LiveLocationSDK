.. title:: LiveLocationSDK

LiveLocationSDK Documentation
=============================

Living Map provides mapping and routing solutions for complex spaces in the form of software development kits (SDKs). This documentation details how to integrate and use the *LiveLocationSDK* which provides functionality for storing and serving the live location of users of the Living Map positioning system within the *LivingMapSDK*.
